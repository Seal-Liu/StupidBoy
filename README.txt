18B20的TI驱动包括三个文件
DS1820_CONFIG.c
DS1820_DRIVER.c
DS1820_DRIVER.h


DS1820_CONFIG.c
------------------------------------------
主要是相关的IO口配置和初始化
包括IO的方向设置，IO的初始化，以及延时函数的参数设定
与硬件相关的操作在这个程序中已经完成了。


DS1820_DRIVER.c
------------------------------------------
包含18B20的时序操作，数据的读写，这个文件与硬件无关，可以跨平台移植
文件中还包含一个温度读取的DEMO程序，应用层程序可以参考实例进行程序编写。


DS1820_DRIVER.h
------------------------------------------
18B20的接口文件
包括硬件层和应用层使用的接口都在文件内部



每个文件都有详细的注释，详细细节请见注释。
